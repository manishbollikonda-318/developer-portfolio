# Manish Bollikonda - Creative Developer Portfolio

An immersive, highly-animated developer portfolio built with the latest web technologies to provide a cinematic, premium user experience.

![Portfolio Preview](./public/preview.png)

## 🚀 Tech Stack

*   **Framework:** [Next.js 16 (App Router)](https://nextjs.org/)
*   **UI Library:** [React 19](https://react.dev/)
*   **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
*   **Animations:** [Framer Motion 12](https://www.framer.com/motion/)

## ✨ Key Features

*   **Cinematic Hero Sequence:** A frame-by-frame scrolling animation leveraging HTML5 Canvas overlayed with smooth typography transitions.
*   **Deep Blended Theme:** A cohesive `#090a0f` deep navy-black background seamlessly transitioning the visual depth across sections.
*   **Performance First:** Built with modern CSS custom properties (`@theme`) and React Server Components for blistering fast load times.
*   **Interactive Navigation:** A responsive, glassmorphic floating navigation bar that dynamically highlights the active section.
*   **Resume Quick-Actions:** A polished floating action button for instant interaction to preview or download a PDF resume.

## 🛠️ Local Development

To run this project locally, clone the repository and install dependencies:

```bash
# Install dependencies
npm install

# Run the local development server
npm run dev
```

Open [https://developer-portfolio-nu-sable.vercel.app/] with your browser to see the result.

## 📁 Project Structure

*   **/src/app**: Next.js App Router layout and main page orchestration.
*   **/src/components**: Reusable UI blocks including:
    *   `ScrollyCanvas.tsx`: Master component for the scrolling image frame animation.
    *   `Overlay.tsx`: Interactive text elements synced precisely to scroll progress.
    *   `Projects.tsx`: Showcase grid for featured works with glassmorphic cards.
    *   `About.tsx` & `Contact.tsx`: Professional summary and connection points.
    *   `Navbar.tsx` & `ResumeButton.tsx`: Global navigation overlays.
